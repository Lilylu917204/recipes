## About the project

---

![homepage](/src/images/homepage.png)

#### This is a Recipe app.

#### you will find:

- Five types of meal:
  - breakfast
  - lunch
  - dinner
  - snack
  - teatime
- Search multiple recipes.
- Saved the recipes you love in the list.
- Get recipe's details and ingredients.
